# Contributing to Farm AI & B2B Marketplace

Thank you for your interest in contributing! This project aims to empower farmers through technology, and your help is highly valued.

## 🌈 Our Code of Conduct
Please be respectful and professional in all interactions. We are building this for a community that deserves our best efforts.

## 🛠️ How Can I Contribute?

### Reporting Bugs
- Use the [Issues](https://github.com/Tanish-Chavir/FOXES/issues) tab to report bugs.
- Include steps to reproduce and system details.

### Suggesting Enhancements
- Open an issue with the tag "enhancement".
- Describe the feature and why it would be useful for farmers.

### Code Contributions
1. Check existing issues or open a new one to discuss your idea.
2. Ensure you follow the existing code style (Modular components for React, Clean Express routes).
3. Update the `README.md` if you add new features.
4. Write clear commit messages.

## 🧪 Testing
- Ensure your changes don't break existing navigation.
- Verify that the Marketplace iframe still loads correctly after UI changes.
- Check API compatibility if you modify the backend.

## 📮 Contact
If you have questions, feel free to reach out to the project maintainers via GitHub.

Let's build the future of Indian agriculture together! 🌾
